#MVP
- As a developer, I want my site to use valid and semantic markup, so that employers will love me.
- As a reader, I want to find out more about the vision and mission of the organization to see if they are credible.
- As a reader, I want the site to look reasonable, so that I can read it on any device.
- As a developer, I want the CSS to be organized in SMACCS formatting and the scripts in MVC structure, so I can easily navigate and read the code.
- As a donor, I want requests to have information about the requester and a story about their struggle, so I can pick who I would like to donate to.
- As a developer, I want to declare templates for gift requests, so that I can display the requests easier - grid view.

# Stretch Goals User Stories Stage 1
- As a requester, I want a clear and easy to use form page, so I can easily input my request (on local device).
- As a donor, I want to be able to filter requests by selected categories to fulfill requests that are inline with my organization.
- As a donor, I want a page to confirm my total selections before committing to a claim.
- As a developer, I want to declare templates for gift requests, so that I can display the requests easier - list view toggle.

# Stretch Goals User Stories Stage 2
- As a requester, I want a clear and easy to use form page, so I can easily input my request (on remote server).
- As a donor, I want a stats page to be able to see the impact that I have made.
- As an admin, I want a page where we can easily retrieve claim statuses to distribute gift cards.
